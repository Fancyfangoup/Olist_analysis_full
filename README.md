# Olist_analysis_full
Kaggle Olist 电商数据分析实战 | Python | JupyterLab | 网易求职项目
## 项目背景
基于 Kaggle 公开的巴西电商 Olist 数据集，完成从数据清洗、探索性分析到业务洞察的全流程数据分析，旨在提升电商业务运营效率与用户体验。

## 技术栈
- Python (Pandas, Matplotlib, Seaborn)
- JupyterLab
- 数据分析与可视化

## 项目结构
- `Olist_analysis_full.ipynb`: 完整分析代码（含数据清洗、可视化、业务结论）
- `巴西电商Olist分析书.pdf`: 项目分析报告（业务解读与结论）

## 项目亮点
1.  完整复现电商数据分析全流程，覆盖用户、订单、支付、物流等核心维度
2.  输出可落地的业务洞察与优化建议，贴合互联网电商业务场景
3.  代码规范清晰，注释完整，便于阅读与复用
